Comandos usados en esta prueba:
    git init
    git status
        Muestra el estado de git
    git add "Archivo"
    git add .
        Agregar todos los archivos
    git commit
    git commit -m "Mensaje"
        Evitar entrar al editor de codigo
    git log
    git checkout "Archivo"
    git diff "Archivo"
        Muestra diferencia del archivo
    .gitignore
        Crear archivo para no agregar alguna 
        carpeta/archivo a mi entorno de trabajo
    git branch
    git branch "Version/Rama"
        Crear una nueva version alternativa
    git checkout "Version/Rama"
        Cambiar de version/rama

Subir proyecto a GitHub
    git remote add origin "Url_github"
        Url_github: https://github.com/Eriqmsp/git-test.git
    git push -u origin "Version/Rama"
