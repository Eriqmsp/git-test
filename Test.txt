Comandos usados con Git:
    git init
    git status
        Muestra el estado de git
    git add "Archivo"
        Agregar archivo indicado
    git add .
        Agregar todos los archivos
    git commit
    git commit -m "Mensaje"
        Evitar entrar al editor de codigo VIN
    git log
    git config -- global user.email "email" 
        Configurar email del usuario
        git config -- global user.name "nombre" 
        Configurar nombre del usuario
    git checkout "Archivo"
    git diff "Archivo"
        Muestra diferencia del archivo
    .gitignore
        Crear archivo para ignorar un carpeta 
        o archivo en mi entorno de trabajo
    git branch
    git branch "Version/Rama"
        Crear una nueva version alternativa
    git checkout "Version/Rama"
        Cambiar de version/rama
